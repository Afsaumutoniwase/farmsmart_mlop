How to Use

1. Upload Image: Click the upload area and select a plant image.

2. Predict: Click "Predict Disease" to analyze the image.

3. View Results: See the top prediction with confidence score.

4. Retrain: Upload new images to improve the model.

Retraining Tips

Training: Upload 10+ images of the same disease.

Validation: Upload 5+ different images of the same class.

Class Name: Use clear labels like Tomato_New_Disease.

Auto Sorting: Images will be organized into folders.
